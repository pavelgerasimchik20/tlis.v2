# Прайс-лист (тротуарная плитка)

Файл **`pricelist-trotuarnaya-plitka.docx`** содержит:

- логотип (SVG `logo-dark.svg` → PNG при сборке);
- блок **TLIS** / **Производственная компания**;
- **разделы** (структура по аналогии с прайсом УДМС — матрицы «размер × цвет»):
  1. **Тротуарная плитка «Color Mix»** — строки: наши форматы (20×10×8, 20×10×6, паркет 48×12×8, мегаполис 60×30×8, тактильные 20×10×9, комплект «Новый город» на 1 м²); **колонки** — цвета каталога Color Mix (как в УДМС), без колонки «Закат» (не выпускается); мок. **50 руб/м²** в каждой ячейке.
  2. **Тротуарная плитка на сером цементе** — те же строки по размерам; **колонки**: серая, чёрная, жёлтая, коричневая, красная, зелёная (как в УДМС); мок.: серая **20 руб/м²** (−30 к базе Color Mix), остальные **40 руб/м²**.
  3. **Одноцветная** (белая, синяя и др. вне таблицы на сером цементе) — мок. **40 руб/м²**.
  4. **Бордюры** БР100.20.8 и БР100.30.15 — мок. **10 руб/п.м.**

## Пересборка

```bash
cd angular-app
npm run generate:pricelists
```

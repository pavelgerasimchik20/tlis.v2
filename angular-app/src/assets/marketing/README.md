# Маркетинговые материалы TLIS

Эта папка содержит все маркетинговые материалы компании TLIS для использования в рекламе, печати и цифровых каналах.

## Структура папок

### `/logo` - Логотипы

Содержит логотип компании в различных форматах:

- **logo-color.svg** - Цветной вариант логотипа (для темного фона)
- **logo-white.svg** - Белый вариант на прозрачном фоне (для темного фона)
- **logo-dark.svg** - Темный вариант (для светлого фона)
- **logo-export.html** - Инструмент для экспорта логотипа в PNG различных размеров

#### Использование logo-export.html:

1. Откройте файл в браузере
2. Выберите нужный вариант логотипа
3. Нажмите кнопку с нужным размером (200x200, 400x400, 800x800, 1200x1200)
4. Изображение автоматически скачается в формате PNG

### `/business-cards` - Визитки

- **business-card.html** - Готовая визитка для печати

#### Использование:

1. Откройте `business-card.html` в браузере
2. Нажмите "Печать визитки" для печати
3. Или "Экспорт в PNG" для сохранения цифровой версии
4. Стандартный размер визитки: 90mm × 50mm

**Рекомендации для печати:**
- Используйте плотную бумагу (300 г/м²)
- Печать в высоком качестве (300 DPI)
- Убедитесь, что принтер поддерживает точные размеры

### `/banners` - Баннеры

- **banner-stretch.html** - Баннер-растяжка для наружной рекламы

#### Использование:

1. Откройте `banner-stretch.html` в браузере
2. Размер баннера: 3000px × 500px (соотношение 6:1)
3. Используйте "Экспорт в PNG" для получения изображения
4. Или "Печать баннера" для печати

**Рекомендации для печати:**
- Разрешение: минимум 150 DPI для наружной рекламы
- Формат: PNG или PDF
- Размер: 3000px × 500px (можно масштабировать)

## Технические требования

### Для печати:
- Формат: PDF или PNG
- Разрешение: минимум 300 DPI
- Цветовая модель: CMYK (для офсетной печати) или RGB (для цифровой печати)

### Для веб-использования:
- Формат: PNG, SVG или JPG
- Разрешение: 72-150 DPI
- Цветовая модель: RGB

## Примечания

- Все SVG файлы можно масштабировать без потери качества
- Для экспорта в PNG используйте инструменты браузера или специальные онлайн-конвертеры
- Рекомендуется использовать векторные форматы (SVG) для логотипов везде, где это возможно

# Генерация PNG и JPEG логотипов

## Способ 1: Через браузер (РЕКОМЕНДУЕТСЯ) ⭐

Самый простой способ - использовать HTML файл:

1. **Откройте файл `generate-png-jpeg.html` в браузере**
2. Нажмите кнопку **"Сгенерировать все логотипы (PNG + JPEG)"**
3. Файлы автоматически начнут скачиваться
4. **Переместите все скачанные файлы в папку `marketing/logo/`**

### Что будет создано:
- **PNG:** 200×200, 400×400, 800×800, 1200×1200 пикселей
- **JPEG:** 200×200, 400×400, 800×800, 1200×1200 пикселей
- **Варианты:** Цветной, Белый, Темный
- **Всего:** 24 файла (12 PNG + 12 JPEG)

---

## Способ 2: Через Node.js (для продвинутых)

Если у вас установлен Node.js и вы хотите автоматизировать процесс:

1. **Установите зависимости:**
   ```bash
   npm install canvas
   ```

2. **Запустите скрипт:**
   ```bash
   node generate-images.js
   ```

3. Файлы будут созданы прямо в папке `marketing/logo/`

**Примечание:** Установка `canvas` может потребовать дополнительных системных библиотек в зависимости от вашей ОС.

## Рекомендации по использованию:

- **PNG** - используйте для веб-сайтов, когда нужна прозрачность или высокое качество
- **JPEG** - используйте для печати или когда нужен меньший размер файла
- **200×200** - для иконок, аватаров
- **400×400** - для средних изображений на сайте
- **800×800** - для больших изображений, превью
- **1200×1200** - для печати высокого качества

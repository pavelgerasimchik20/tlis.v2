ИНСТРУКЦИЯ ПО ОБНОВЛЕНИЮ FAVICON:

✅ Файл переименован в favicon-32x32.png

Теперь нужно:

1. ОСТАНОВИТЕ dev server (если запущен):
   - Нажмите Ctrl+C в терминале

2. ОЧИСТИТЕ КЭШ БРАУЗЕРА:
   - Chrome/Edge: Ctrl+Shift+Delete → выберите "Изображения и файлы в кэше" → Очистить
   - Или откройте сайт в режиме инкогнито: Ctrl+Shift+N

3. ПЕРЕЗАПУСТИТЕ dev server:
   npm start

4. ОТКРОЙТЕ САЙТ С ЖЕСТКОЙ ПЕРЕЗАГРУЗКОЙ:
   - Ctrl+F5 (жесткая перезагрузка)
   - Или Ctrl+Shift+R

5. ПРОВЕРЬТЕ:
   - Откройте напрямую: http://localhost:4200/favicon-32x32.png
   - Должен отображаться логотип TLIS

Если все еще не видно:
- Закройте все вкладки с сайтом
- Очистите кэш браузера полностью
- Откройте сайт заново
